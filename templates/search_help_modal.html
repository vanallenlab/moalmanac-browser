Search help

The search function supports queries for feature names, attribute names & values, diseases, therapies, and predictive implications. Queries may be combined; queries across different categories will be combined with a boolean AND operation (e.g., Melanoma AND Erlotinib) while queries within the same category will be combined with a boolean OR operation (e.g., Melanoma OR Glioma).

Attribute searches require specifying both the attribute name and the desired attribute value, separated by a colon (:). For example, to search for all assertions referencing a Knockdown experiment where the technique used was shRNA, enter "technique:shRNA" and select the Knockdown option.
